# Hiro Marker untuk AR.js

## Download Marker

**Link resmi Hiro Marker:**
https://raw.githubusercontent.com/AR-js-org/AR.js/master/data/images/hiro.png

## Cara Menggunakan

1. Download gambar Hiro Marker dari link di atas
2. Print di kertas putih (ukuran minimal 10x10 cm)
3. Atau tampilkan di layar komputer/tablet
4. Pastikan marker tidak terlipat atau rusak
5. Gunakan di tempat dengan cahaya cukup

## Preview Marker

Marker Hiro adalah gambar kotak hitam dengan pola khusus di tengahnya.

Pastikan marker:
- ✅ Kontras tinggi (hitam putih jelas)
- ✅ Tidak blur atau rusak
- ✅ Ukuran cukup besar (minimal 8x8 cm)
- ✅ Di permukaan datar
- ✅ Tidak ada bayangan atau pantulan

## Alternatif

Jika tidak bisa print, Anda bisa:
1. Buka link marker di laptop/tablet lain
2. Tampilkan fullscreen
3. Arahkan kamera HP ke layar tersebut
4. Pastikan tidak ada pantulan cahaya di layar
